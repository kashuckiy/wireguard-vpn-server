# WireGuard VPN Server

Мінімальний, автоматизований та надійний VPN сервер на базі WireGuard для вашого VPS (Debian/Ubuntu). Цей проект містить набір bash-скриптів та зручну обгортку `Makefile` для розгортання та повного управління VPN сервером.

Підходить для таких хмарних провайдерів:
- **Oracle Cloud**
- **AWS (Amazon Web Services)**
- **DigitalOcean**
- **Hetzner**
- **Будь-який інший VPS з Debian/Ubuntu**

---

## 🌟 Можливості

- 🚀 **Швидке встановлення:** Автоматичне встановлення пакетів, увімкнення IP forwarding, генерація ключів та налаштування NAT.
- 📱 **Зручне управління клієнтами:** Легке додавання клієнтів із **генерацією QR-кодів** для швидкого підключення на мобільних пристроях.
- 🛡️ **Надійний розподіл IP:** Запобігання конфліктам IP-адрес шляхом автоматичного визначення наступної вільної адреси (без перехрещень).
- 🧹 **Повне видалення клієнтів:** Видалення клієнта не лише з активного інтерфейсу, але й акуратне очищення конфігураційних файлів (`wg0.conf`) для уникнення їх відновлення після перезавантаження сервера.
- 💾 **Резервне копіювання (Backup & Restore):** Створення та відновлення резервних копій усіх конфігурацій та ключів в один клік.

---

## 🛠 Попередні вимоги

1. **ОС:** Debian або Ubuntu.
2. **Права доступу:** Користувач з правами `sudo` або `root`.
3. **Firewall (Брандмауер):** Переконайтеся, що порт **51820 / UDP** відкритий у налаштуваннях безпеки вашого хмарного провайдера (наприклад, Security Lists в Oracle Cloud або Security Groups в AWS) та локальному файрволі (UFW / iptables).

## 📦 Встановлення сервера

Клонуйте репозиторій на ваш VPS:

```bash
git clone https://github.com/kashuckiy/wireguard-vpn-server.git
cd wireguard-vpn-server
```

Запустіть процес встановлення:

```bash
make install
```

> **Що робить `make install`?**  
> 1. Оновлює списки пакетів та встановлює `wireguard`, `qrencode`, `curl`.  
> 2. Вмикає `net.ipv4.ip_forward` для маршрутизації трафіку.  
> 3. Генерує приватний та публічний ключі сервера в каталог `/etc/wireguard/`.  
> 4. Автоматично знаходить ваш основний мережевий інтерфейс і створює конфігурацію `/etc/wireguard/wg0.conf` з правилами NAT (MASQUERADE).  
> 5. Активує та запускає системну службу `wg-quick@wg0`.

## 👥 Управління клієнтами

### Додавання клієнта

```bash
make add-client
```
Після команди скрипт запитає ім'я клієнта (наприклад, `iphone` або `laptop`).  
**Результат:** 
- Буде згенеровано `.conf` файл конфігурації у папці `/etc/wireguard/clients/`.
- Файл конфігурації буде виведено в термінал.
- **QR-код** для підключення з мобільних пристроїв відобразиться безпосередньо в консолі!

### Видалення клієнта

```bash
make remove-client
```
Введіть ім'я клієнта, якого бажаєте видалити.  
**Результат:** Клієнта буде негайно відключено, його конфігураційний файл буде видалено з `/etc/wireguard/clients/`, а запис у файлі `/etc/wireguard/wg0.conf` буде чисто стерто, що гарантує його невідновлення після перезавантаження сервера.

## ⚙️ Управління сервером

Команди для базового управління статусом VPN-служби за допомогою `systemctl` та `wg`:

- `make status` — виводить поточний статус активних з'єднань, пірів (клієнтів), і переданий трафік (обгортка над `sudo wg`).
- `make start` — запускає інтерфейс WireGuard.
- `make stop` — зупиняє інтерфейс WireGuard.
- `make restart` — перезапускає інтерфейс.
- `make logs` — перегляд логів WireGuard у реальному часі (обгортка над `journalctl`).

## 💾 Резервне копіювання та відновлення

Для захисту від втрати ключів та конфігурацій реалізовано просту систему бекапів.

**Створення бекапу:**
```bash
make backup
```
Створює архів `wireguard_backup_YYYY-MM-DD_HH-MM-SS.tar.gz` в папці `./backups/`. Цей архів містить весь вміст `/etc/wireguard`.

**Відновлення з бекапу:**
```bash
make restore
```
Скрипт запитає шлях до файлу бекапу. Після його надання сервер розпакує ключі до `/etc/wireguard` та автоматично перезавантажить службу VPN для негайного застосування.

## 🔍 Перевірка роботи VPN

Після підключення клієнта ви можете перевірити свій поточний зовнішній IP-адрес. Введіть на клієнтському пристрої (або VPS):

```bash
curl ifconfig.me
```

Він повинен збігатися з публічним IP вашого VPS сервера.

## 📁 Структура проекту

- `Makefile` — Основний інтерфейс для управління.
- `scripts/install_wg.sh` — Скрипт встановлення бази WireGuard та налаштування мережі.
- `scripts/add_client.sh` — Скрипт-генератор клієнтських конфігурацій та QR кодів, інтелектуальний розподіл IP адрес мережі `10.8.0.0/24`.
- `scripts/remove_client.sh` — Безпечне видалення клієнтів з сервера та конфігурацій.
- `scripts/backup.sh` — Архівація каталогів з ключами.
- `scripts/restore.sh` — Відновлення ключів з архіву.
- `backups/` — Директорія для автоматично створених архівів.
